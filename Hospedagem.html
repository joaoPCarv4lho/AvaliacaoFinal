O Que é Hospedagem?

Hospedagem refere-se ao serviço que permite armazenar e disponibilizar um site na internet. Quando você cria um site, ele é composto por arquivos, como:


  • Imagens
  • HTML
  • CSS
  • Javascript
entre outros elementos.


Toda construção de páginas web se baseia basicamente nas linguagens HTML, CSS E JS. Esses arquivos precisam estar armazenados em um servidor para que possam ser acessados e visualizados pelos usuários da Internet. sempre disponível e acessível.


Esse serviço envolve um custo, um aluguel de espaço geralmente mantido por uma empresa especializada em serviços de hospedagem na web. Essas empresas possuem servidores de alta capacidade e conectados à internet de alta velocidade, garantindo que o site esteja

DNS - O que é?

O DNS (Domain Name System – Sistema de nome de domínio) converte nomes de domínio legíveis por humanos (por exemplo, www.amazon.com) em endereços IP legíveis por máquina (por exemplo, 192.0.2.44).


Ele que vai apontar para os servidores corretos quando você estiver utilizando serviços de hospedagem. Em outras palavras, o domínio que o serviço de hospedagem te deu, vai ser referenciado por meio do endereço de ip, e o domínio que você adquirir para esse site, vai redirecioná-lo para o mesmo dominio. Isso se chama controle de tráfego.
